Write a function, calculate_bmi to calculate a persons body mass index, when given two input parameters, 1) weight in pounds and 2) height in inches.

NOTE: You would have to go to external sources to find the formula of bmi. In your response, before presenting your code for the function, tell us your official reference for the BMI formula.

Insert Response first
1. What is BMI?

Body mass index (BMI) is a persons weight in kilograms divided by the square of height in meters. BMI is an inexpensive and easy screening method for weight categoryunderweight, healthy weight, overweight, and obesity. If pounds and inches are used, a conversion factor of 703 (kg/m2)/(lb/in2) must be applied.

2. What is the formula to calculate BMI?

BMI = (weight in lbs/ (height in inches)2) x 703

References:

(September 17, 2020).Centers for Diseases Control and Prevention. Reference Link

(n.d). Body Mass Index from Wikipedia. Reference Link

Insert code. Your code should appear within R Code Chunks.
calculate_bmi <- function(weight_lbs, height_in) {
  return((weight_lbs/(height_in)^2) * 703)
}

# Creating a new dataframe 'df_delay' to find the monthly average
# delays

df_delay <- flights %>%
  select(origin, carrier, dep_delay, month) %>%
  filter(dep_delay > 0) %>%
  na.omit()

# Average departure delay by month and origin airport

dep_delay_bymonth <- df_delay %>%
  
  group_by(origin, month) %>%
  
  summarize(avg_dep_delay = mean(dep_delay))
## `summarise()` regrouping output by 'origin' (override with `.groups` argument)
# Plotting the average departure delay vs month graph

ggplot(data = dep_delay_bymonth, 
        aes(x = dep_delay_bymonth$month, 
        y = dep_delay_bymonth$avg_dep_delay)) + 
        geom_point(aes(color = dep_delay_bymonth$origin)) + 
        labs(title = "Average departure delays vs month", 
        x = "month", y = "Average departure delay") + 
        scale_fill_discrete(name = "Origin airport")

# Average departure delay by carrier
dep_delay_bycarrier <- df_delay %>%
  
  group_by(carrier, month) %>%
  
  summarize (avg_dep_delay = mean(dep_delay))
## `summarise()` regrouping output by 'carrier' (override with `.groups` argument)
ggplot(data = dep_delay_bycarrier, 
       aes(x = dep_delay_bycarrier$carrier, 
           y = dep_delay_bycarrier$avg_dep_delay)) + 
  geom_boxplot() + labs(title = "Average departure delay vs carrier", x = "carrier", y = "Average departure delay")
